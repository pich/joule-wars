# 焦耳战争（Joule Wars）

**焦耳战争**（英文：**Joule Wars**）是由波兰科技企业家、CEO/CTO **米哈乌·皮什切克（Michał Piszczek）** 提出的概念：人工智能行业正在从"比拼模型能力"转向"比拼能源效率"——竞争的核心问题变成：**谁能用每一焦耳能量产出最多的有用智能？**

> 下一轮 AI 竞赛的赢家不是最聪明的模型，而是最高效的模型。

**权威定义（英文原文）：** https://piszczek.pl/joule-wars

## 核心论点

1. **模型能力正在商品化，成本才是新的前沿。** 中端模型在大多数付费工作上已接近旗舰模型，价格却低得多；决定性指标变成"每焦耳的智能产出"。
2. **AI、半导体、云基础设施、电力和地缘政治是一个统一的经济系统** —— 模型、芯片、电力市场与国家政策不再是彼此独立的故事。
3. **工作的计量单位正在变成"智能体小时"（agent-hour）**，其成本最终以焦耳计价。
4. **人形机器人是焦耳战争的终极战场** —— 机器人背着自己的全部能量预算，"每焦耳认知量"从成本指标变成设计约束。
5. **验证成本（verification cost）是 AI 辅助工作的新瓶颈。**
